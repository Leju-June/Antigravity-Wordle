# Antigravity-Wordle
﻿Windows 환경에서 Vmware Workstation Pro를 설치하고, kali linux 환경에서 앱 개발을 진행함.
antigravity ai를 사용함, 사용한 모델은 gemini 3.1 pro
언어 - 파이썬
프레임 워크 - Reflex ( 직관성이 높고 해석하기 쉽기 때문에 수정이 자유로움 )
개발할 앱 - Wordle
